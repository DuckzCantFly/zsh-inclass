# zsh-inclass

To install this config use.

### Step 1

Make sure git is installed.

```bash
sudo dnf install git
```

### Step 2

Move old `.zshrc`.

```bash
mv ~/.zshrc ~/.zshrc.old

```

### Step 3

Download new `.zshrc`.

```bash
wget "https://raw.githubusercontent.com/DuckzCantFly/zsh-inclass/main/.zshrc"
```

### Step 4

Move in to zsh shell.

```bash
zsh
```

### (Optional) Step 5

Change your user shell to zsh. 

```bash
chsh /usr/bin/zsh
```
